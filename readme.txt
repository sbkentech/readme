readme
본 프로그램은 Gaussian Distribution을 사용하여 P(X|C_k)를 계산합니다.
이를 위해 각 클래스 C_k(여기서는 0과 1)에 대해 다음을 정의합니다.
P(X|C_k): 클래스 C_k에서 데이터 X의 조건부 확률로 가우시안 함수 모델링
P(C_k):클래스 C_k의 사전 확률로, 학습 데이터에서 해당 클래스의 비율로 계산하였다.
본 연산과정은 bayse.py에서 진행되었다.

본 프로그램은 학습으로 사용하는 데이터가 크게 많지 않고 이상치보다는 정상값이 훨씬 더 많기 때문에 MAP를 이용하여 naive bayes classifier을 제작하였습니다.


주어진 train / testing csv파일을 이용하여 진행되었으며, 하이퍼 파라미터는 ϵ입니다. ϵ값을 변화시키는 실험은 epsilon.py에서 진행되었습니다.
